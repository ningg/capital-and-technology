# capital-and-technology
资本、科技，股票交易的思考。

赚钱，可以从 5 个要素出发：土地、劳动、资本、技术、管理。

其中，不同的要素，特性也不同，特别是杠杆率不同：

* 土地、劳动：线性
* 资本、技术、管理：指数

这里，我们专注资本和技术，并将其应用在股票交易，预期将获取应得的回报。
